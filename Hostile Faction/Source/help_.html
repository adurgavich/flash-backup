  Basic Rain and Lightning effect

FOR: FLASH 8 (AS 2.0)


To use the Rain or Lightning:
Copy and paste Rain clip and/or Lightning clip into your project.
( Adjustable settings are inside the Rain and Lightning clips )



Create 600x400 movie at around 60 FPS.





RAIN FALL:

1. Draw or find a graphic of a rain drop (with transparency support such as .png format).

2. Create raindrops movie clip in library:
    - Place raindrops in upper-left corner on frame1 main stage.

3. Create raindrop_graphic movie clip in library:
    - Place raindrop.png image inside raindrop_graphic (align: Y-bottom, X-center).
    - Place raindrop_graphic inside raindrops.
    - Set the raindrop_graphic instance name to "raindrop0".

4. Inside raindrops, place this "generator" code in frame1 actions:

rainDrop_amount=32;

for(n=0; n <= rainDrop_amount; n++){
duplicateMovieClip(raindrop0, "raindrop0"+n, n);
};


5. Inside raindrop_graphic, place this basic "rain" code in frame1 actions:

fall_speed=72;
span_left=0;
span_right=600;
restartY=450;

ini_Y_coord=-50+(Math.random()*450);
_y=ini_Y_coord;
ini_X_coord=span_left+Math.random()*span_right;
_x=ini_X_coord;
function preSPAWN(){
sizeMIN=4;
sizeMAX=48-sizeMIN;
rand_scale = sizeMIN+ ( Math.random()*(Math.random()*sizeMAX) );
_width=rand_scale;
_height=(_width*2.3);
rand_Y_spd= (fall_speed + (Math.random()*fall_speed))/10;
rand_Y_end= restartY+Math.random()*16;
};
preSPAWN();

function reSPAWN(){
preSPAWN();
rand_Y_coord=-(Math.random()*50);
_y=rand_Y_coord;
rand_X_coord=span_left+Math.random()*span_right;
_x=rand_X_coord;
};

function aLIVE(){
_y+=(rand_Y_spd)*(0.5+rand_scale/6);
};

function control_FUNCTIONS(){
if(_y>=rand_Y_end){
reSPAWN();
}else{
aLIVE();
};
};
onEnterFrame=control_FUNCTIONS;

6. Run movie(CTRL+Enter) to test rain effect.
... and adjust (tweak properties/settings) as necessary.





LIGHTNING:

1. Draw or find a graphic of a lightning bolt (with transparency support such as .png format).

2. Create lightning movie clip in library:
    - Place lightning in upper-left corner on frame1 main stage.
    - Place lightning.png image inside lightning (align: Y-bottom, X-center).

3. Inside lightning, place this basic "lightning" code in frame1 actions:

chance=75;
span_left=0;
span_right=580;
size_min=64;
size_max=820;

function spawn(){
rand_N=-1+Math.random()*2;
if(rand_N<0){
_xscale=-(size_min+Math.random()*size_max);
_yscale=-(_xscale)+(Math.random()*size_min);
}else if(rand_N>0){
_xscale=size_min+(Math.random()*size_max);
_yscale=_xscale+(Math.random()*size_min);
};
randALPHA_fade=2.5+(Math.random()*8);
_x=span_left+Math.random()*span_right;
_y=-60+Math.random()*20;
_rotation=-8+(Math.random()*16);
randLIGHTglow=50+(Math.random()*130);
_alpha=randLIGHTglow/1.2;
};

function active(){
_alpha -= randALPHA_fade/1.2;
};

onEnterFrame=function(){
_chanceA=Math.random()*Math.random()*chance;
_chanceB=Math.random()*2000;
if( _chanceA >= _chanceB ){
spawn();
}else{
active();
};
};

4. Run movie(CTRL+Enter) to test lightning effect.
... and adjust (tweak properties/settings) as necessary.
